# On-Call Guide

The on-call rotation changes every Monday at 9am. The current on-call engineer is always listed in the #oncall Slack channel topic.

On-call engineers are expected to acknowledge any critical alert within 15 minutes. If the primary on-call does not respond within 15 minutes, the alert automatically escalates to the secondary on-call.

After each on-call shift, the engineer should write a short handover note covering any ongoing issues, so the next person has full context.
